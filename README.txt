For information on how to build Instantbird from the source code, see:

    http://wiki.instantbird.org/Instantbird:Compiling

Bug reports and feature suggestions are welcome in Bugzilla:

    https://bugzilla.instantbird.org/

If you have a question about using or developing Instantbird, and
can't find the solution on http://wiki.instantbird.org, you can try
asking your question on IRC in the chatroom #instantbird on the
irc.mozilla.org server.

You can download nightly development builds from the Instantbird
download server.  Keep in mind that nightly builds, which are used for
testing, may be buggy.  Instantbird nightlies can be found at:

    http://ftp.instantbird.com/instantbird/nightly/latest-trunk/
